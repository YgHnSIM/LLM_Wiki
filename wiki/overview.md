# LLM Wiki Overview

이 위키는 대규모 언어 모델(LLM)과 관련된 지식을 체계적으로 정리하고 학습하기 위한 지식 베이스입니다.

## 🎯 현재 상태
- **초기화 완료**: 새로운 지식을 수집할 준비가 되었습니다.
- **주요 주제**: LLM 역사, 아키텍처, 학습 방법론, 주요 논문 등.

## 🚀 시작하기
1. `raw/` 디렉토리에 새로운 소스(논문, 기사 등)를 추가합니다.
2. LLM에게 `ingest` 요청을 하여 위키 페이지와 퀴즈 카드를 생성합니다.
3. 생성된 `[[index]]`를 통해 지식의 지도를 탐색합니다.
