# LLM Wiki Overview

이 위키는 대규모 언어 모델(LLM)과 관련된 지식을 체계적으로 정리하고 학습하기 위한 지식 베이스입니다.

## 핵심 지도

현재 위키의 출발점은 [[클로드 섀넌]], [[앨런 튜링]], [[조지타운-IBM 기계번역 시연]], [[프랭크 로젠블랫]], [[노엄 촘스키]], [[버나드 위드로]]와 [[마션 호프]]의 여섯 질문입니다. 섀넌의 계보는 언어를 어떻게 확률적으로 예측할 수 있는지 묻고, 튜링의 계보는 언어 행동이 지능의 증거가 될 수 있는지 묻습니다. 조지타운-IBM의 계보는 한 언어의 의미를 다른 언어로 자동 변환할 수 있는지 묻고, 로젠블랫의 계보는 기계가 예시에서 직접 학습할 수 있는지 묻습니다. 촘스키의 계보는 언어가 어떤 [[통사 구조]]를 가지며, 그 구조를 어떻게 형식적으로 설명할 수 있는지 묻습니다. 위드로-호프의 계보는 신경망이 실제 통신과 신호 처리 문제에 안정적으로 배치될 수 있는지 묻습니다.

[[N그램 모델]]은 언어를 실제 사용 빈도와 조건부 확률로 다루는 단순한 모델이며, [[마르코프 가정]]을 통해 계산 가능성을 얻는 대신 긴 문맥을 잃습니다. 이 한계는 [[데이터 희소성]], [[스무딩]], [[카츠 백오프]], [[분산 표현]]으로 이어졌고, 자세한 전환 과정은 [[N그램에서 신경망 언어 모델로]]에서 볼 수 있습니다.

[[튜링 테스트]]는 언어가 문맥 이해, 지식 사용, 추론, 적응, 창의성을 함께 요구한다는 점을 보여 줍니다. 현대 [[대규모 언어 모델]]은 이 질문을 현실화했지만, 동시에 인간다움만으로는 충분하지 않은 [[벤치마크]]와 안전성 평가의 필요성을 드러냅니다. 자세한 논의는 [[튜링 테스트와 현대 LLM 평가]]에서 볼 수 있습니다.

[[기계번역]]은 언어 AI가 실제 문제를 해결하려는 초기 응용이었습니다. [[규칙 기반 번역]]과 [[사전 기반 접근]]은 가능성을 보여 주었지만, [[모호성 해소]]와 [[지식 공학 병목]]의 한계를 드러냈습니다. 이 흐름은 [[병렬 말뭉치]]와 [[신경망 기계번역]]으로 이어졌고, 자세한 분석은 [[규칙 기반 번역에서 신경망 기계번역으로]]에서 볼 수 있습니다.

[[퍼셉트론]]은 데이터에서 학습하는 신경망의 기초를 세웠습니다. 단층 모델의 [[선형 분리 가능성]] 한계와 [[XOR 문제]]는 [[다층 퍼셉트론]], [[역전파]], [[딥러닝]]으로 이어지는 연구 의제를 만들었습니다. 자세한 흐름은 [[퍼셉트론에서 딥러닝으로]]에서 볼 수 있습니다.

[[월터 피츠]]와 [[워런 매컬러]]의 [[맥컬록-피츠 뉴런]]은 퍼셉트론보다 앞선 신경망적 사고의 출발점입니다. 뉴런을 [[논리 게이트]]처럼 추상화한 이 모델은 [[마음의 계산 이론]], [[사이버네틱스]], [[프로그램 내장형 컴퓨터]], [[연결주의]]를 잇는 통로가 되었고, 자세한 흐름은 [[맥컬록-피츠에서 현대 신경망으로]]에서 볼 수 있습니다.

[[MADALINE]]은 [[ADALINE]], [[LMS 알고리즘]], [[적응 필터]]를 통해 신경망을 실제 통신·음성 신호 처리 문제로 옮긴 사례입니다. 이 계보는 [[음성 활동 감지]], [[에코 제거]], [[특징 공학]], [[적응 신호 처리]]로 이어지며, 현대 음성 기반 언어 AI의 전단을 이해하는 데 중요합니다. 자세한 흐름은 [[MADALINE에서 현대 음성 AI로]]에서 볼 수 있습니다.

[[통사론]]과 [[변형생성문법]]은 언어를 단어열이 아니라 위계적 구조와 [[재귀]]를 가진 체계로 다룹니다. [[문맥자유문법]], [[촘스키 위계]], [[파싱]]은 초기 [[계산언어학]]과 컴퓨터과학에 형식적 도구를 제공했고, 이 흐름이 현대 [[대규모 언어 모델]]의 구조 학습 논쟁으로 이어지는 과정은 [[촘스키에서 대규모 언어 모델로]]에서 볼 수 있습니다.

스키너의 *Verbal Behavior* 비평은 촘스키 계보를 행동주의 언어 이론과 더 직접적으로 연결합니다. [[기능적 분석]], [[자극 통제]], [[강화]], [[맨드와 택트]], [[오토클리틱]]은 스키너의 설명 도구이고, [[자극의 빈곤]]과 언어 창의성 논점은 이 설명의 한계를 드러냅니다.

[[존 설]]의 중국어 방 논쟁은 튜링 계보를 [[강한 AI]], [[약한 AI]], [[의도성]], [[기호 접지 문제]], [[생물학적 자연주의]]로 확장합니다. 이 보강은 현대 [[대규모 언어 모델]] 평가에서 기능적 성능과 실제 이해의 문제를 구분해야 한다는 관점을 강화합니다.
