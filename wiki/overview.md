# LLM Wiki Overview

이 위키는 대규모 언어 모델(LLM)과 관련된 지식을 체계적으로 정리하고 학습하기 위한 지식 베이스입니다.

- **주요 주제**: [[정보 이론]], [[N그램 모델]], [[튜링 테스트]], [[기계번역]], [[퍼셉트론]], [[딥러닝]], [[대규모 언어 모델]].

## 핵심 지도

현재 위키의 출발점은 [[클로드 섀넌]], [[앨런 튜링]], [[조지타운-IBM 기계번역 시연]], [[프랭크 로젠블랫]]의 네 질문입니다. 섀넌의 계보는 언어를 어떻게 확률적으로 예측할 수 있는지 묻고, 튜링의 계보는 언어 행동이 지능의 증거가 될 수 있는지 묻습니다. 조지타운-IBM의 계보는 한 언어의 의미를 다른 언어로 자동 변환할 수 있는지 묻고, 로젠블랫의 계보는 기계가 예시에서 직접 학습할 수 있는지 묻습니다.

[[N그램 모델]]은 언어를 실제 사용 빈도와 조건부 확률로 다루는 단순한 모델이며, [[마르코프 가정]]을 통해 계산 가능성을 얻는 대신 긴 문맥을 잃습니다. 이 한계는 [[데이터 희소성]], [[스무딩]], [[카츠 백오프]], [[분산 표현]]으로 이어졌고, 자세한 전환 과정은 [[N그램에서 신경망 언어 모델로]]에서 볼 수 있습니다.

[[튜링 테스트]]는 언어가 문맥 이해, 지식 사용, 추론, 적응, 창의성을 함께 요구한다는 점을 보여 줍니다. 현대 [[대규모 언어 모델]]은 이 질문을 현실화했지만, 동시에 인간다움만으로는 충분하지 않은 [[벤치마크]]와 안전성 평가의 필요성을 드러냅니다. 자세한 논의는 [[튜링 테스트와 현대 LLM 평가]]에서 볼 수 있습니다.

[[기계번역]]은 언어 AI가 실제 문제를 해결하려는 초기 응용이었습니다. [[규칙 기반 번역]]과 [[사전 기반 접근]]은 가능성을 보여 주었지만, [[모호성 해소]]와 [[지식 공학 병목]]의 한계를 드러냈습니다. 이 흐름은 [[병렬 말뭉치]]와 [[신경망 기계번역]]으로 이어졌고, 자세한 분석은 [[규칙 기반 번역에서 신경망 기계번역으로]]에서 볼 수 있습니다.

[[퍼셉트론]]은 데이터에서 학습하는 신경망의 기초를 세웠습니다. 단층 모델의 [[선형 분리 가능성]] 한계와 [[XOR 문제]]는 [[다층 퍼셉트론]], [[역전파]], [[딥러닝]]으로 이어지는 연구 의제를 만들었습니다. 자세한 흐름은 [[퍼셉트론에서 딥러닝으로]]에서 볼 수 있습니다.
