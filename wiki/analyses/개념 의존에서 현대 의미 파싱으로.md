# 개념 의존에서 현대 의미 파싱으로

[[개념 의존 이론]]의 역사적 의미는 특정 원시 행동 목록보다 넓다. 이 이론은 [[자연어 이해]]가 [[파싱]]만으로 끝나지 않으며, 문장의 표층 형식과 독립적인 [[표준 의미 표현]]이 필요하다는 문제를 분명히 제기했다.

첫 번째 계승점은 사건과 역할의 구조화다. [[원시 행동]]과 [[개념적 격]]은 "누가 무엇을 누구에게 했는가"를 계산 가능한 구조로 표현하려 했다. 현대 [[의미역 레이블링]]과 [[프레임 의미론]]도 술어와 참여자의 역할을 찾는다는 점에서 같은 질문을 이어받는다.

두 번째 계승점은 그래프형 의미 표현이다. 개념 의존 구조는 사건과 참여자를 네트워크로 연결했다. 현대 [[추상 의미 표현]]과 [[지식 그래프]]는 다른 표기법과 학습 방법을 쓰지만, 텍스트 의미를 구조화된 그래프로 바꾸려는 목표를 공유한다.

세 번째 계승점은 이야기와 상식 추론이다. [[MARGIE]]가 의역과 질의응답을 다루었다면, [[SAM]]은 [[스크립트]]로 생략된 사건을 보완했고 [[PAM]]은 계획과 목표를 이용해 행동의 의도를 해석했다. 이 흐름은 현대 에이전트형 [[대규모 언어 모델]]의 상태 추적, 도구 호출, 계획 검증 문제와 연결된다.

가장 큰 단절점은 학습 방식이다. 개념 의존 시스템은 사람이 원시 행동, 격, 어휘 사상, 추론 규칙을 직접 작성해야 했고 이 때문에 [[지식 획득 병목]]에 부딪혔다. 현대 신경망 시스템은 대규모 데이터에서 표현을 학습해 커버리지를 넓혔지만, 명시적 구조와 설명 가능성은 약해질 수 있다.

따라서 현대적 과제는 둘 중 하나를 선택하는 것이 아니다. 데이터 기반 모델의 범용성과 구조화된 의미 표현의 설명 가능성을 결합해, 다양한 언어 표현을 공통 의미 구조로 모으면서도 실제 추론과 검증에 쓸 수 있는 표현을 만드는 것이 핵심이다.
