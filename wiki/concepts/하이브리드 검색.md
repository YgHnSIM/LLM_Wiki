# 하이브리드 검색

하이브리드 검색(Hybrid Search)은 [[BM25]]나 [[TF-IDF]] 같은 희소 검색과 [[조밀 벡터]] 기반 [[의미 검색]]을 결합하는 방식이다. 두 방식은 서로 다른 강점을 갖기 때문에 함께 쓰면 검색 품질이 좋아질 수 있다.

희소 검색은 사용자가 입력한 정확한 단어, 고유명사, 숫자, 코드, 희귀 용어를 잘 잡아낸다. 반면 조밀 검색은 단어가 직접 겹치지 않아도 의미가 가까운 문서를 찾는 데 강하다.

[[검색 증강 생성]]에서는 하이브리드 검색이 자주 쓰인다. 먼저 희소 검색과 조밀 검색으로 후보 문서를 넓게 모으고, 재랭킹 또는 LLM 기반 판단으로 답변에 쓸 근거를 좁히는 구조다.

하이브리드 검색은 [[벡터 공간 모델]]의 유산을 잘 보여 준다. 고전적 TF-IDF와 현대 임베딩은 표현 방식은 다르지만, 모두 텍스트를 벡터화하고 유사성을 계산한다는 같은 문제 설정을 공유한다.
