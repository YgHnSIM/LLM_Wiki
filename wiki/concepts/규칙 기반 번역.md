# 규칙 기반 번역

규칙 기반 번역(Rule-Based Machine Translation, RBMT)은 사람이 만든 사전과 문법 규칙을 이용해 한 언어를 다른 언어로 변환하는 방식이다. [[조지타운-IBM 기계번역 시연]]의 시스템은 [[사전 기반 접근]]과 [[구문 규칙]]을 결합한 초기 규칙 기반 번역 사례였다 [[003_1954_조지타운-IBM_기계번역]].

이 방식의 장점은 해석 가능성이다. 어떤 단어가 어떤 후보로 번역되었고, 어떤 규칙이 어순을 조정했는지 비교적 추적할 수 있다. 초기 연구자들은 언어 지식을 명시적으로 부호화하면 번역 문제를 계산 절차로 다룰 수 있다고 보았다 [[003_1954_조지타운-IBM_기계번역]].

한계는 확장성이다. 사전에 없는 단어, 예상하지 못한 문장 구조, 관용 표현, 복잡한 종속절, [[모호성 해소]]가 필요한 문맥에서는 쉽게 실패한다. 또한 언어쌍과 영역이 늘어날수록 사람이 만들어야 할 규칙과 사전이 급격히 증가한다 [[003_1954_조지타운-IBM_기계번역]].
