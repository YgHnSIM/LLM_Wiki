# 머신러닝

머신러닝(Machine Learning)은 사람이 모든 규칙을 직접 작성하기보다 데이터에서 패턴을 학습하게 하는 AI 접근법이다. 현재 소스는 [[규칙 기반 번역]]에서 [[신경망 기계번역]]으로의 전환을 지식 공학에서 머신러닝으로 이동한 더 넓은 흐름의 사례로 설명한다 [[003_1954_조지타운-IBM_기계번역]].

[[조지타운-IBM 기계번역 시연]] 같은 초기 시스템은 사람이 사전과 규칙을 직접 구축해야 했다. 이 방식은 해석 가능하지만 확장성이 낮았고, [[지식 공학 병목]]을 만들었다 [[003_1954_조지타운-IBM_기계번역]].

현대 기계번역과 [[대규모 언어 모델]]은 대량의 데이터에서 언어 패턴과 표현을 학습한다. 이 전환은 더 높은 성능과 일반화를 가능하게 했지만, 해석 가능성 감소와 데이터 의존성이라는 새로운 문제도 낳았다 [[003_1954_조지타운-IBM_기계번역]].
