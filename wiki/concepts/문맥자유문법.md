# 문맥자유문법

문맥자유문법(Context-Free Grammar, CFG)은 구 구조를 재작성 규칙으로 나타내는 형식문법이다. 현재 소스는 문맥자유문법을 [[변형생성문법]]의 단순화된 형식이자 초기 [[파싱]]의 표준 도구로 설명한다.

문맥자유문법은 문장을 명사구와 동사구 같은 하위 구성요소로 분해한다. 이런 규칙은 [[통사 구조]]를 트리 형태로 표현할 수 있게 해 주며, [[유한상태 모델]]보다 중첩 구조를 더 잘 다룬다.

이 형식은 자연어 처리뿐 아니라 프로그래밍 언어와 컴파일러에도 중요하다. 컴파일러가 소스 코드를 파싱해 추상 구문 트리를 만들 때, 문맥자유문법의 계보에 있는 도구를 사용한다.
