# 지식 획득 병목

지식 획득 병목(Knowledge Acquisition Bottleneck)은 시스템에 필요한 개념, 규칙, 사례, 추론 지식을 사람이 직접 입력해야 해서 확장이 어려워지는 문제다. [[개념 의존 이론]]의 주요 한계로 등장한다.

개념 의존 시스템은 동사를 [[원시 행동]]으로 매핑하고, 문장 성분을 [[개념적 격]]에 배정하며, 행동의 전제 조건과 결과를 규칙으로 작성해야 했다. 새로운 도메인이나 새로운 사건 유형이 나타날 때마다 이 작업을 반복해야 했다.

이 병목은 [[상징주의 AI]]의 넓은 한계와 연결된다. 명시적 구조와 설명 가능성은 강점이지만, 모든 지식을 수작업으로 작성하면 열린 세계의 자연어를 감당하기 어렵다.

현대 [[머신러닝]]과 [[대규모 언어 모델]]은 데이터에서 표현을 학습해 이 문제를 완화하지만, 구조적 설명 가능성과 안정적 추론을 항상 보장하지는 않는다.
