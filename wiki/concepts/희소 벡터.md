# 희소 벡터

희소 벡터(Sparse Vector)는 대부분의 차원 값이 0인 벡터다. [[TF-IDF]] 문서 벡터는 전체 어휘의 각 단어를 차원으로 삼기 때문에 차원 수는 크지만, 한 문서가 실제로 사용하는 단어는 일부에 불과하다.

[[SMART 시스템]]은 0이 아닌 용어와 가중치만 저장하는 방식으로 희소 벡터의 메모리 문제를 줄였다. 이는 오늘날 역색인과 희소 검색 구조의 기본 직관과 연결된다.

희소 벡터는 정확한 용어 매칭, 희귀 고유명사, 설명 가능성에 강하다. 반면 동의어와 문맥을 처리하는 데 약해 [[어휘 불일치]] 문제가 생긴다. 이 한계를 보완하기 위해 [[조밀 벡터]] 기반 [[의미 검색]]이 발전했다.
