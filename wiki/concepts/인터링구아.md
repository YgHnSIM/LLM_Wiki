# 인터링구아

인터링구아(Interlingua)는 여러 언어나 여러 표층 표현을 잇는 공통 의미 표현이다. [[개념 의존 이론]]에서는 자연어 문장을 직접 비교하기보다 공통 의미 구조로 번역해 저장하고, 필요하면 다시 자연어로 생성하는 방식을 지향했다.

인터링구아의 장점은 의역과 번역을 같은 문제로 다룰 수 있다는 점이다. "John gave Mary a book"과 "Mary received a book from John"이 같은 ATRANS 구조로 변환되면, 서로 다른 표현이 같은 사건임을 알 수 있다.

이 관점은 [[기계번역]]과도 연결된다. 원천 언어 문장을 공통 의미 표현으로 바꾼 뒤 목표 언어 문장을 생성한다는 발상은 규칙 기반 번역과 의미 기반 번역 연구에서 중요한 설계였다.
