# 템플릿 기반 응답

템플릿 기반 응답(Template-Based Response)은 미리 작성한 문장 틀에 사용자 입력의 일부를 넣어 응답을 생성하는 방식이다. [[ELIZA]]는 [[패턴 매칭]]으로 키워드를 찾은 뒤, 해당 패턴에 연결된 응답 템플릿을 선택했다.

이 방식은 사용자의 단어를 다시 포함하기 때문에 개인화된 대화처럼 보인다. 하지만 실제로는 의미 관계를 이해하기보다 문자열 일부를 추출하고 치환하는 절차다. 소스는 이 간극이 ELIZA의 설득력과 한계를 동시에 만든다고 설명한다.

템플릿 기반 응답은 반복을 피하기 위해 여러 문장 틀을 순환하거나 무작위로 선택할 수 있다. 이러한 변주는 시스템이 더 자연스럽고 덜 기계적으로 보이게 한다.

현대 [[챗봇]]과 고객지원 플로우에서도 템플릿은 여전히 쓰인다. 다만 [[대규모 언어 모델]]은 고정 템플릿보다 훨씬 다양한 문장을 생성하므로, 사용자가 내부 이해를 과대평가할 위험도 더 커진다.
