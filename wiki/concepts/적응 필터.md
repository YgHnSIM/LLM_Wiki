# 적응 필터

적응 필터(Adaptive Filter)는 입력 신호와 환경 변화에 맞춰 필터 계수를 자동으로 조정하는 신호 처리 시스템이다. 현재 소스는 [[MADALINE]]과 [[LMS 알고리즘]]을 적응 필터링의 초기 신경망적 사례로 설명한다.

고정 필터는 회선 조건, 잡음, 장비 상태가 바뀌면 성능이 떨어질 수 있다. 적응 필터는 관찰된 오차를 이용해 계수를 계속 수정하므로 [[에코 제거]], 잡음 억제, 채널 등화 같은 문제에 적합하다.

현대 음성 AI에서도 적응 필터의 관점은 중요하다. [[대규모 언어 모델]]이나 음성 인식 모델이 언어 처리를 하기 전에, 마이크 신호에서 잡음과 에코를 줄이는 안정적 음향 전처리가 필요하기 때문이다.
