# 의미 파싱

의미 파싱(Semantic Parsing)은 자연어 문장을 계산 가능한 의미 구조로 변환하는 과정이다. [[파싱]]이 문법 구조를 분석한다면, 의미 파싱은 사건, 참여자, 관계, 조건, 질의 구조를 표현한다.

[[개념 의존 이론]]은 의미 파싱의 초기 기호주의 형태로 볼 수 있다. 문장을 ATRANS, PTRANS, MTRANS 같은 [[원시 행동]]과 [[개념적 격]]의 네트워크로 바꾸어 의역 인식과 추론을 가능하게 하려 했다.

현대 의미 파싱은 [[의미역 레이블링]], [[프레임 의미론]], [[추상 의미 표현]], 데이터베이스 질의 변환, 도구 호출용 슬롯 추출 등으로 확장되었다. 신경망 모델은 이 변환을 수작업 규칙이 아니라 데이터에서 학습하는 경우가 많다.

[[검색 증강 생성]]과 에이전트형 [[대규모 언어 모델]]에서도 의미 파싱은 중요하다. 사용자의 다양한 표현을 공통 질의, 의도, API 호출, 지식 그래프 질의로 바꾸어야 하기 때문이다.
