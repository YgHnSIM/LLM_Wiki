# 대규모 언어 모델

대규모 언어 모델(Large Language Model, LLM)은 대량의 텍스트로 학습해 다음 토큰 예측, 대화, 요약, 번역, 코딩, 추론 보조 등을 수행하는 언어 모델이다. [[N그램 모델]]과 마찬가지로 주어진 문맥에서 다음 항목의 확률 분포를 추정한다는 문제 설정을 공유하지만, 단순한 카운트 테이블 대신 신경망과 [[분산 표현]]을 사용한다 [[001_1948_클로드 섀넌_N그램 모델]].

[[튜링 테스트]]의 관점에서 LLM은 언어 AI의 역사적 목표가 상당 부분 현실화된 사례다. GPT-4, Claude, Gemini 같은 모델은 제한된 조건의 대화에서 인간 응답과 구별하기 어려운 수준의 언어 행동을 보일 수 있다 [[002_1950_앨런 튜링_튜링 테스트]].

그러나 LLM의 유창성은 곧바로 진정한 이해나 의식을 증명하지 않는다. [[중국어 방 논증]]이 제기하는 것처럼, 적절한 언어 출력과 의미 이해 사이에는 여전히 철학적·기술적 간극이 있다 [[002_1950_앨런 튜링_튜링 테스트]].

현대 LLM 평가는 인간다움만으로 충분하지 않다. 사실성, 환각, 편향, 도구 사용, 장기 작업 수행, 안전성, 사용자 과신 가능성 같은 기준이 함께 필요하다 [[002_1950_앨런 튜링_튜링 테스트]].

기계번역의 관점에서 LLM은 번역을 독립 도구가 아니라 대화형 작업 흐름의 일부로 통합한다. 번역, 요약, 용어 정리, 문체 조정, 문서 비교, [[후편집]]이 하나의 인터페이스 안에서 결합될 수 있다 [[003_1954_조지타운-IBM_기계번역]].

신경망의 관점에서 LLM은 [[퍼셉트론]]의 기본 아이디어를 거대한 규모로 확장한 후손이기도 하다. 내부의 선형층은 입력 표현에 [[가중치]] 행렬과 [[편향]]을 적용하며, 전체 모델은 오류를 줄이도록 매개변수를 반복적으로 조정한다 [[004_1958_프랭크 로젠블랫_퍼셉트론]].
