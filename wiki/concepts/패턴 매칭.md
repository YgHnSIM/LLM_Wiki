# 패턴 매칭

패턴 매칭(Pattern Matching)은 입력에서 미리 정의한 키워드나 구조를 찾아 대응 규칙을 실행하는 방식이다. [[ELIZA]]에서는 사용자의 문장에서 "mother", "dream" 같은 단어를 찾아 적절한 [[템플릿 기반 응답]]을 선택하는 데 쓰였다.

ELIZA의 패턴 매칭은 의미 이해와 다르다. 프로그램은 사용자가 왜 특정 말을 했는지 추론하지 않고, 입력 문자열의 표면적 특징을 기준으로 응답 경로를 선택했다. 하지만 키워드 우선순위가 심리적으로 중요한 주제에 맞춰 설계되었기 때문에, 결과는 통찰처럼 보일 수 있었다.

패턴 매칭은 [[규칙 기반 번역]]이나 초기 [[자연어 처리]] 시스템과도 연결된다. 사람이 규칙을 명시적으로 설계한다는 점에서는 통제 가능하지만, 입력 다양성과 문맥 변화에 취약하고 확장성이 제한된다.

현대 [[대규모 언어 모델]]은 고정된 키워드 규칙만으로 작동하지 않지만, 사용자가 표면적 적절성을 실제 이해로 오해할 수 있다는 문제는 [[ELIZA 효과]]의 형태로 계속 남아 있다.
