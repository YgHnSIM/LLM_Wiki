# 특징 공학

특징 공학(Feature Engineering)은 원시 데이터에서 모델이 사용할 유용한 입력 특징을 사람이 설계하는 작업이다. 현재 소스는 [[MADALINE]] 시대의 신호 처리 시스템에서 특징 공학이 핵심 역할을 했다고 설명한다.

음성 처리 예시에서 엔지니어는 오디오를 주파수 대역별 에너지나 파형 통계로 바꾸어 [[ADALINE]] 유닛에 입력했다. 모델이 원시 파형에서 표현을 자동으로 발견하는 것이 아니라, 사람이 문제에 맞는 표현을 먼저 만들어 주어야 했다.

현대 [[딥러닝]]은 많은 경우 표현을 자동 학습하지만, 특징 공학의 문제의식은 여전히 남아 있다. 입력 표현, 전처리, 데이터 품질, 도메인 지식은 모델 성능과 배치 안정성에 직접 영향을 준다.
